## 🚀 Interactive Documentation (Prototype)

A beginner-friendly interactive documentation website for Mesa.

👉 https://sharmadaivik1308-hue.github.io/mesa_Daivik/

### Features:
- Visual explanation of core concepts (Agent, Model, Step)
- Interactive simulation demonstrating agent-based behavior
- Beginner-friendly learning flow

### Motivation:
Mesa documentation is powerful but can be challenging for new users.
This project aims to improve onboarding by providing a simpler and more interactive learning experience.
